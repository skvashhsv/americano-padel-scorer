# Americano Padel Scorer

Веб-приложение для ведения счета в паделе по правилам "Американо".

## Особенности

- ✅ Ведение счета по правилам "Американо"
- ✅ Автоматическое определение победителя
- ✅ История действий с функцией отмены (Undo)
- ✅ Адаптивный дизайн для мобильных устройств
- ✅ Сохранение состояния в Local Storage
- ✅ Управление с клавиатуры

## Правила "Американо"

1. Игра ведется до установленного количества очков (по умолчанию 15)
2. Можно выиграть гейм только на своей подаче
3. При равном счете играется тай-брейк до 2-х очков разницы
4. Матч играется до установленного количества побед в сетах

## Горячие клавиши

- `1` - добавить очко Team A
- `2` - добавить очко Team B
- `Ctrl+Z` / `Cmd+Z` - отменить последнее действие
- `Ctrl+R` / `Cmd+R` - сбросить матч
- `Escape` - закрыть все окна

## Используемые технологии

- Vue.js 3
- Tailwind CSS
- Font Awesome
- Local Storage API
